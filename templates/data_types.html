Data Types in Python

Python supports several data types to represent different kinds of values. Here are some common data types:

  • Integer: An integer is a whole number in Python, positive or negative, without any fractional component (e.g., 42, -7).
  • Float: A float is a number in Python that has a decimal point or is expressed in scientific notation (e.g., 3.14, -0.001, 1e3
  • String: A string is a sequence of characters in Python enclosed in single, and double quotes (e.g., 'hello', "world")
  • Boolean: A boolean represents a logical value in Python, either True or False

Examples:

Integer:

        
x = 5
y = -10
print(x)  # Output: 5
print(y)  # Output: -10
